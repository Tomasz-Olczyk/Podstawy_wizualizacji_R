# Repozytorium kursu 

## Cel kursu

Zapoznanie Słuchaczy z podstawami technik tworzenia różnych typów wizualizacji danych z wykorzystaniem narzędzi środowiska R. Udział w kursie nie wymaga wiedzy na temat środowiska R, programowania i statystyki.

## Szczegółowe cele kursu

- opanowanie podstaw działania środowiska programistycznego R i programu  RStudio,
- zaznajomienie z popularnymi pakietami języka R służącymi importowaniu i przekształcaniu danych,
- poznanie zagadnień teorii grafiki informacyjnej i jej implementacji w - pakietach środowiska R
- opanowanie technik tworzenia różnego typu statycznych i interaktywnych wizualizacji danych z wykorzystaniem bibliotek ggplot2 i ggiraph

## Program kursu

Program zaprojektowany został z myślą o nauce praktycznych umiejętności pracy z danymi, od ich przygotowania po skuteczną prezentację wizualną. Szczególny nacisk położony jest na rolę wizualizacji w analizie i interaktywnym komunikowaniu informacji w środowisku analitycznym.

1. Podstawy środowiska R i RStudio.
2. Gramatyka grafiki informacyjnej – podstawy.
3. Przygotowywanie i przekształcanie danych.
4. Wizualizacja w komunikowaniu danych.
5. Wykresy interaktywne.
